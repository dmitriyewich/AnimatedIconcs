# NoNameAnimHUD
Анимированные иконки. Пока только: обводка, фист(idle), дигл, м4 дробовик.

Активация: /animhud.

Требования: MoonLoader 026.5+, mimgui, MoonAdditions. (автоскачивания библиотек нет)

Установка: Всё в moonloader
